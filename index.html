ZADANIE DLA GRUPY 1CE

  • git config --global user.email "" - ustalenie adresu e-mail dla aktualnego użytkownika np. git config --global user.email "dawmaj@st.amu.edu.pl"
  • git branch -d "branch" - usuwa gałąź o nazwie branch np. git branch -d "moja_galaz" - usuwam gałąź o nazwie "moja_galaz"
  • git commit -a -m - zapisuje nowo dodane zmiany oraz dodaje komentarz np. git commit -a -m "Dodano nowy komentarz"- dodano nowe zmiany z komentarzem, że dodno nowy komentarz
  • git fetch "remote" "branch" - pobranie zmian z repozytorum remote do gałęzi branch
  • git log --graph --all - wyświetla graficznie historię wszystkich logów w repozytorium
  • git checkout -b "branch" - umożliwia przełączenie na gałąź branch"
  • git diff --staged - pokazuje różnice między ostatnim commitem a wszystkim zastage'owanymi zmianami
  • git remote add "remote_name" "git url" -tworzy zdalne repozytorium o nazwie "remote_name"
